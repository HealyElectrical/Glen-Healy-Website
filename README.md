# Glen-Healy-Website
Website displaying personal projects and resume
This repo hosts my GitHub Pages site.

## Edit content
- Update `index.html` for About/Tools/Education
- Update `assets/data/projects.json` for project cards
- Create a project page in `projects/` per project
- Replace `assets/resume/resume.pdf` with your resume PDF

## Deploy (GitHub Pages)
Settings -> Pages -> Deploy from a branch -> main / root
